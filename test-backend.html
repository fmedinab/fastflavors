🔧 Diagnóstico Backend - Fast Flavors

Verifica la conexión con Google Apps Script y carga los datos iniciales

📋 Instrucciones:

  1. Asegúrate de haber desplegado tu Google Apps Script como Web App
  2. Copia la URL de implementación que termina en /exec
  3. Pégala abajo y prueba la conexión
  4. Si todo funciona, actualiza el archivo js/config.js con esta URL
Nota: Esta URL debe ser la del deployment de tu Apps Script, NO la del editor. Debe terminar en /exec.
